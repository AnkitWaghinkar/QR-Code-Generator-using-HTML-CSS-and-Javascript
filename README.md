# QR Code Generator

## Description
A QR code generator built with HTML, CSS, and JavaScript. It allows users to generate QR codes from text or URLs.

## Features
- Generate QR codes from any text or URL.
- Download generated QR codes.
- Simple and user-friendly interface.

## Technologies Used
HTML, CSS and Javascript
